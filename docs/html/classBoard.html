Detailed Description

template<typename Key, typename T, unsigned int Dimensions>
class Board< Key, T, Dimensions >

A N-dimensional board where put stuff A container of elements that inherit drom type T. These elements can be set over a N-dimensional space in which each cell is identified by a N-index.
